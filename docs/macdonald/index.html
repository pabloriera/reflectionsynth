MacDonald 1978 - Distributed Delay via Linear Chain

Model definition

The delay variable uses the Erlang kernel from equation (7), then the distributed delay is replaced by a finite chain of ODEs for realtime audio.

$$G_a^p(u) = \frac{a^{p+1} u^p}{p!}e^{-au}$$

$$\bar{\tau} = \int_0^{\infty} u\,G_a^p(u)\,du = \frac{p+1}{a}, \qquad a = \frac{p+1}{\bar{\tau}}$$

$$\dot{x}(t) = \frac{\beta\tanh(\gamma z(t)) - x(t)}{\tau}, \qquad z(t) = y_p(t)$$

$$\dot{y}_0 = a(x-y_0), \qquad \dot{y}_j = a(y_{j-1}-y_j), \; j=1,\dots,p$$

Audio is the current state $x(t)$, integrated sample-by-sample with a selectable explicit Euler or RK4 solver.
