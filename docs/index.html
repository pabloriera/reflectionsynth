Reflection function synth studies

This front page routes into two browser instruments built around reflection-function models. Both demos expose the resonator response directly, so you can hear how nonlinear excitation and reflected energy shape the synthesized sound.

Two approaches, one idea. The Schumacher demo explores a nonlinear Volterra formulation with convolution kernel r(t), while the Barjau demo focuses on a woodwind DDE model with editable bore geometry and reflection profile h(t).

Schumacher

McIntyre, Schumacher, Woodhouse

Nonlinear Volterra synthesis with an explicit reflection kernel.

1983

This synth presents the causal solver around the reflection function r(t), the state variable q(t), and the nonlinear coupling map F. It is suited for inspecting how the kernel, excitation, and transfer terms interact over time.

Kernel view Inspect and relate the reflection response to the resulting waveform.
State evolution Track the nonlinear solver variables directly in the browser interface.

Barjau

Woodwind DDE instrument

Reed-bore interaction driven by delay, damping, and reflection control.

1997

This synth focuses on a woodwind model where the reflection function h(t) and the bore profile can be drawn or reset from the physical parameters. It makes the link between tube geometry, delay, and sonic response immediate.

Editable reflection Draw a custom reflection curve and hear the model respond in real time.
Bore geometry Switch or sketch resonator shapes to explore cylindrical, conical, and flared behavior.

About the project. Reflecton gathers browser-based synth experiments that use reflection functions as the central representation of the resonator. Instead of hiding the model behind presets, these demos expose the response functions, delays, and nonlinear controls so the acoustic structure remains audible and visible.

Use this page as the entry point for GitHub Pages, then continue into the instrument that matches the physical model you want to explore.
